# novaXchange – React + Tailwind + Framer Motion

Modern, scalable landing page for novaXchange – a student-first tech upgrade platform in Uganda.

## Tech Stack

- React 18 + Vite
- Tailwind CSS
- Framer Motion
- React Icons
- React Router (ready for future pages)

## Features

- Full responsive, mobile-first design
- Video background hero with rotating animated cards
- Custom cursor (desktop only)
- Scroll reveal animations
- Trade-in value calculator
- Lead capture form (Google Form + WhatsApp integration)
- Floating WhatsApp button with pulse
- No installment plans (removed per requirements)
- Architected for future Web3, admin dashboard, MongoDB

```bash
npm install
npm run dev
